
機能一覧
日時を表示する機能。
N校通学コースの授業時間に合わせ音声通知、デスクトップ通知をする機能。
csvファイルの時間割データを読み取り、今日の時間割を割り出し表示する機能。
上記データを使用しデスクトップ通知に授業内容を表示する機能。

使い方
csvファイルに自身の時間割を、メモ帳などのテキストエディタを使用し打ち込んでください。
csvファイルを簡単に制作するには、エクセルを使用しxlsxファイルに入力後、
名前を付けて保存-ファイルの種類{CSV UTF-8(コンマ区切り)(*.csv)}
で簡単に制作することができます。
CSVファイルのエクセルによる直接編集は正しく読み取れなくなるため絶対にしないでください。

chime*.mp3という音声データはaudioタグを使用して再生しています。ご自由に差し替えてご利用ください。
mp3でなくてもaudioタグに対応していれば問題ありません。
1は授業開始時、2は授業終了(除3の条件時)時、3は全ての授業終了時(15:40のみ)に再生されます。

注意事項
時間割データは再読み込みされません。曜日が変わった時点で再読み込みをしてください。
通知についてはページ読み込み時から1秒ごとに監視しているため、1秒以下の遅れが発生しますが仕様です。
choromeで開いた場合通知が表示されない事象を確認しています。firefoxでは問題なく動作しています。
